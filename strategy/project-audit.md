# Project Audit

## BayonHub
- **Name**: BayonHub
- **One-line description**: Cambodia-first tech student opportunity hub for internships, freelance work, jobs, and portfolio-building opportunities.
- **Role**: Founder-engineer
- **Stack**: React, Vite, TypeScript, Express, Prisma, PostgreSQL, Redis, Tailwind CSS, Docker, Cloudflare/Vercel direction
- **Timeline**: 2024–Present
- **Honest outcome statement**: BayonHub began as a Khmer-first classifieds prototype, then pivoted fully into a tech-student opportunity platform. Phase 0 now establishes student, employer, and admin identity plus the opportunity/application data foundation. The retired marketplace remains in Git history, but the active public case study shows only the current product direction.

## Svaeng Yul
- **Name**: Svaeng Yul
- **One-line description**: Khmer-narrated self-study video platform for Cambodian students in AI engineering, data science, cybersecurity, and software engineering.
- **Role**: Product builder and full-stack engineer
- **Stack**: Next.js, React, TypeScript, Supabase, PostgreSQL, Bunny Stream, Anthropic API, Tailwind CSS
- **Timeline**: 2026-Present
- **Honest outcome statement**: Svaeng Yul previously explored a medical question-bank direction. The active private product is now a technology-learning platform with demo/live data modes, a course publishing workflow, progress tracking, protected video direction, and a lesson-scoped AI assistant. It remains a private launch candidate until real infrastructure, Khmer content, policies, and human verification are complete.
