# Project Audit

## BayonHub
- **Name**: BayonHub
- **One-line description**: Cambodia-first tech student opportunity hub for internships, freelance work, jobs, and portfolio-building opportunities.
- **Role**: Founder-engineer
- **Stack**: React, Vite, TypeScript, Express, Prisma, PostgreSQL, Redis, Tailwind CSS, Docker, Cloudflare/Vercel direction
- **Timeline**: 2024–Present
- **Honest outcome statement**: BayonHub began as a Khmer-first classifieds prototype with working marketplace flows. It is now pivoting into a tech-student opportunity platform. The old prototype remains useful public proof of engineering execution, while the current private source moves toward student/employer auth, profiles, opportunities, applications, and admin-curated listings.
