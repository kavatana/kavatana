# Project Audit

## BayonHub
- **Name**: BayonHub
- **One-line description**: A Khmer-first classifieds marketplace for Cambodia.
- **Role**: Founder & Product Engineer
- **Stack**: Nuxt 3, TypeScript, NestJS, PostgreSQL, Redis, Tailwind CSS, Docker, Cloudflare, Vercel
- **Timeline**: 2024–Present
- **Honest outcome statement**: Full buyer-to-seller marketplace functionality is live, featuring search, listing detail pages, post/draft/publish listing flows, user accounts, buyer-seller messaging, and a dedicated trust system (phone verification, seller response rates, last-seen indicators). The build demonstrates end-to-end product architecture and execution, focusing on core foundations rather than superficial features.
