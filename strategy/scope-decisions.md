# Scope Decisions

- **Writing section**: OUT OF SCOPE for v1. Gate condition: minimum 3 publishable pieces must exist before this is added.
- **Future builds section**: IN SCOPE.
  - Rules: compact, high-level, no fake project cards, no timelines, framed as direction not delivery.
  - Label: "What I'm building toward."
  - Content: Marketing Websites, Full-Stack Web Apps, Admin Dashboards, SaaS MVPs, AI Chatbot Products, AI Agent Tools.
- **CMS**: OUT OF SCOPE for v1
- **Backend/API**: OUT OF SCOPE unless contact form requires it (use Resend or Formspree)
- **One flagship project only**: BayonHub. By design, not by limitation.
- **Contact method**: contact form using Resend or Formspree (to be confirmed)
- **Analytics**: Plausible (privacy-first, lightweight)
