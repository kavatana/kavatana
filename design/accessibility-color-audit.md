# Accessibility Color Audit

Verify and record contrast ratios for all text/bg color pairs.

- **Primary Text (`--color-text-primary`) on Background (`--color-bg`)**: PASS (> 14:1)
- **Secondary Text (`--color-text-secondary`) on Background (`--color-bg`)**: PASS (WCAG AA compliant, ~5:1)
- **Primary Text on Surface-1 (`--color-surface-1`)**: PASS (> 13:1)
- **Accent (`--color-accent`) on Background (`--color-bg`)**: PASS (> 4.5:1)
- **Accent (`--color-accent`) on Surface-1 (`--color-surface-1`)**: PASS (> 4.5:1)

*All body text passes WCAG AA (4.5:1). All large text passes WCAG AA (3:1).*
