# Fallback Rules

Behavior for every optional field when absent:

- `screenshots`: section is hidden, not rendered with empty state
- `liveUrl`: external link button is hidden
- `repoUrl`: GitHub link is hidden
- `nextSteps`: section is hidden
- `callout`: callout block is hidden
- `endYear`: display as "2024–Present"
- `featuredOrder`: sort to end of list
