# Storage Conventions

All content lives in `/content/` as JSON files loaded at build time.

- `/content/config.json` → `SiteConfig`
- `/content/capabilities.json` → `CapabilityGroup[]`
- `/content/build-directions.json` → `BuildDirection[]`
- `/content/timeline.json` → `TimelineEntry[]`
- `/content/projects/bayonhub.json` → `Project`
